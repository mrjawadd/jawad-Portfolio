**Description:**  
A modern, responsive personal portfolio website built with **Flask** and **SQLite** to showcase projects, skills, and contact information.  
**Key Features:**  
- Dynamic portfolio sections for projects, skills, and contact  
- Contact form integrated with email  
- Mobile-friendly and responsive design  
**Tech Stack:** Python, Flask, HTML, CSS, SQLite  
**Potential Improvements:** Add blog section or deploy with user authentication for admin.  
**GitHub:** [Flask Portfolio](https://github.com/mrjawadd/jawad-Portfolio)
